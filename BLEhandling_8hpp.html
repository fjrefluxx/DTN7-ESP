Detailed Description

This file contains the helper functions to interact with the nimble GAP/GATT functions.

Note
For the BLE-CLA to work, the following settings must be made in menuconfig: CONFIG_BT_ENABLED = y CONFIG_BT_NIMBLE_ENABLED = y
